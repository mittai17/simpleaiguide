ML has four major types. Each type solves different problems.

## ‚≠ê Type 1 ‚Äî Supervised Learning (Most Important in ML)

**[Supervised learning](https://en.wikipedia.org/wiki/Supervised_learning)** uses **labeled data**.

**Example:**

| Image | Label |
| :--- | :--- |
| Cat photo | Cat |
| Dog photo | Dog |

### Used for:

-   Email spam detection
-   Disease prediction
-   Face recognition
-   Credit card fraud detection

### Why it‚Äôs powerful:

It learns the relationship between `input ‚Üí output`.

## ‚≠ê Type 2 ‚Äî Unsupervised Learning

Learns from **[unlabeled data](https://en.wikipedia.org/wiki/Unsupervised_learning)**. Finds hidden patterns.

### Examples:

-   Customer segmentation
-   Product grouping
-   Document clustering

### Algorithms:

-   K-Means
-   Hierarchical Clustering
-   PCA

## ‚≠ê Type 3 ‚Äî Semi-Supervised Learning

Mix of:

-   Few labeled samples
-   Many unlabeled samples

**Used in:**

-   YouTube video tagging
-   Facial recognition
-   Speech labeling

## ‚≠ê Type 4 ‚Äî Reinforcement Learning (Learning by Reward)

Agent takes actions ‚Üí gets **[reward](https://en.wikipedia.org/wiki/Reinforcement_learning)** ‚Üí learns.

**Used in:**

-   Self-driving cars
-   Robotics
-   Game AI
-   Navigation
